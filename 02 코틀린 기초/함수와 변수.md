# 함수와 변수
## 함수   
### void 함수 

```kt
fun method(a: Int, b: Int) {
    println(a + b)
}
```

### 반환형 함수(문)  
```kt
fun method(a: Int, b: Int): Int {
    return if(a > b) a else b
}
```

### 반환형 함수(식1)

```kt
    return if(a > b) a else b
}
```
