클래스와 프로퍼티
==================
코틀린의 클래스는 자바 클래스에 비해 적은 양의 코드로 원하는 작업을 수행할 수 있다.     
   
**자바 Person**
```java
public class Person {           // 자바 클래스 
    private final String name;  // 프로퍼티 
    
    public Person(String name) {
        this.name = name;
    }
    
    public String getName() {
        return name;
    }
    
}
```
