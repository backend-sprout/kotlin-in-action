선택과 표현 처리: enum과 when
============================
> 코틀린의 구성요소 중 when에 대해서 알아보자    

코틀린의 when은 자바의 swtich를 대치하되 훨씬 더 강력한 기능을 제공한다.      
그리고 when을 알아보면서 코틀린에서 enum을 선언하는 방법과 `스마트 캐스트`에 대해서도 알아보자  

# enum 클래스 정의   

```kt
enum class Color {
    RED, ORANGE, YELLOW, GREEN, BLUE, INDIGO, VIOLET
}
```
